## DiffWind是什么
DiffWind是一个量化投资系统，主要包括基于指标的筛选模型（案例学习部分因为不成熟并没有开源）。

## DiffWind如何使用
DiffWind使用的数据源：  
**新浪：**  
股票行业分类数据  
**雪球：**  
主要财务指标历史数据（下载近10+年数据）  
股本结构历史数据（下载近10+年数据）  
财务报表历史数据（下载近10+年数据）  
**东方财富：**  
融资融券历史数据  
**通达信：**  
日线数据（下载近20年数据）  

**使用步骤：**
1. 安装数据库：PostgreSQL 9.x/10.x  
2. 执行src/main/sqlscripts下的SQL脚本文件，创建表和索引  
3. DiffWind初次执行时需准备历史数据，初始化数据库(只需执行一次，以后不需要再执行)：  
DiffWindFirstTime  
4. 执行筛选模型：  
DiffWindMain
